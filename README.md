# Pokemon Type Calculator WinUI 3

WinUI 3 implementation of the [Pokemon Type Calculator](https://github.com/iyarashii/PokemonTypeCalculator). This is a simple desktop application that calculates the effectiveness of a move against a Pokémon based on the type of the move and the type of the Pokémon. It is written in C# and uses WinUI 3 and .NET 9.
